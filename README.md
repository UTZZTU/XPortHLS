# XPortHLS

XPortHLS is a version-aware migration agent framework for migrating XRT/Vitis HLS applications into native AVED projects.

Competition target:

**XPortHLS: Version-Aware Migration of XRT-Based HLS Applications to Native AVED Projects**

Current implementation status:

- No model training.
- AI API will be used through a model adapter.
- Deterministic components handle scanning, IR, contracts, templates, validation and trace.
- LLM components handle ambiguity, cross-file reasoning, log diagnosis and repair planning.

Current local environment:

- Ubuntu 24.04
- Vivado/Vitis 2025.1
- AVED package not currently present
- V80 board not currently installed
- This repository starts with a stub AVED 2025.1 platform pack.
